HTML5 e CSS

HTML 5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da internet. É a quinta versão da linguagem HTML.

Elementos Semânticos Usado Descrição
header done Usei o header em todas as páginas para mostrar o título com o nome da cidade
nav done Utilizei o nav para fazer o menu de navegação entre as páginas html
main done Utilizei o main para colocar o conteúdo específico à página
article close Não usado
section done Usei o section para separar os conteúdos no main, como parágrafos, imagens, tabelas, etc
aside close Não usado
footer done Usei o footer para colocar o meu nome, número, nome da universidade e ano.

CSS

CSS (Cascading Style Sheets) é um mecanismo para adicionar estilos a uma página web

Seletores CSS Usado Descrição
universal done Usei o seletor universal para mudar a fonte de todas as páginas
tipo done Usei o seletor tipo para alinhar texto e mudar cores (por exemplo o background)
classe done usei o seletor de classe para mudar coisas mais específicas como as cores dos fieldsets
atributo close Não usado
pseudo-classe done usei o seletor de pseudo-classe para fazer efeitos de hover sobre a zona de navegação por exemplo
pseudo-elemento close Não usado
